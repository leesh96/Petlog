안드로이드 어플 개발
데이터베이스는 주로 MySQL, MariaDB, SQLite 사용하는데,
안드로이드에서 바로 데이터베이스에 접근 불가
웹서버 구축 후 연동으로 접근 -> PHP(MySQL, 아파치, Servlet 이용
SQLite - 소규모 데이터 관리 사용에 적합

파이어베이스란?
구글에서 만든 통합 앱 플랫폼
웹과 모바일 개발에 필요한 기능을 제공하는 서비스
백엔드 개발을 통해 서버를 따로 설계, 구현하지 않고 프론트엔드 개발에 집중할 수 있도록 해줌.
주요기능 : 실시간 데이터베이스, 간편한 사용자인증, 클라우드 저장소, 호스팅, 앱 테스트와 수익 창출 등등

우리 어플에 가장 효율적인 방법으로 구축하면 될 것 같음.

기능
1. 일기 기능 : 내 반려동물이 오늘 어떤 감정을 느낀것 같았는지 선택(동물그림 + 표정 + 색상/MOODA 앱 참고), 텍스트로 일기 작성, 사진 첨부 가능, 배경색글씨체 등등 커스터마이징 가능, 날짜별로 분류 가능하게(팔레토 앱 참고)
2. 펫스타그램 : 인스타그램과 유사 - 게시글올리기 (사진필수), 좋아요, 댓글, 사용자태그, 해쉬태그(->해쉬태그 기반 검색 가능해야함), 위치태그(->위치태그 기반 검색가능해야함), 등등 어느정도 까지 구현할 수 있을까?
3. TalkTalk(산책로, 물품나눔) : 산책로 추천, 물품 나눔, 게시판? 기능 4, 6번 묶자
4. 마이펫 : 정보입력, 일정관리(생일, 병원방문 간단하게, 디데이 리마인더 알림 정도로만 간단하게) 기능 3, 5번 묶자
