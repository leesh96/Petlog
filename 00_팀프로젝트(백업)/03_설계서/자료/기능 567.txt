기능5(설정)
역할:어플사용설정하기
입력:설정버튼클릭
출력:오른쪽 설정 메뉴바 출력
기능5.1(로그아웃)
역할:계정로그아웃
입력:로그아웃 버튼 클릭
출력:로그아웃 후 로그인 화면 출력
기능5.2(버전정보)
역할:어플버전정보 보여주기
입력:버전정보 메뉴 클릭
출력:버전정보 출력 팝업
기능5.3(알림)
역할:알림켜고 끄기
입력:알림버튼클릭
출력:알림설정 메뉴바 출력
	기능5.3.1(댓글알림) - 내가 쓴 글에 달린 댓글
	역할:댓글알림 켜고끄기
	입력:댓글 토글 온/오프
	출력:토글 위치에 맞게 설정 변경
	기능5.3.2(게시물알림??) - 구독한 사람 게시물 알림?
	기능5.3.3(좋아요알림??) - 좋아요 눌리면 알림
	기능5.3.4(구독알림??) - 누가 날 구독하면 알림?
기능5.4(소리)
역할:소리 켜고 끄기
입력:소리 토글 온/오프
출력:토글 위치에 맞게 설정 변경
기능5.5(진동)
역할:진동 켜고 끄기
입력:진동 토글 온/오프
출력:토글 위치에 맞게 설정 변경
기능5.6(회원정보관리)
역할:회원정보보기
입력:회원정보 관리 메뉴 클릭
출력:회원정보 창 출력
	
기능5.7(공지사항)
역할:개발자 공지사항 보기
입력:공지사항 메뉴 클릭
출력:공지사항 목록 출력
	기능5.7.1(공지사항보기)
	역할:개발자 공지사항 보기
	입력:목록에서 공지 선택
	출력:공지내용출력

역할:어플 사용자 계정 생성
입력:
출력:
이메일, 이름, 닉네임, 생년월일, 비밀번호 받아서 가입

입력:로그아웃된 상태에서 어플 실행 시 뜬다
출력:
기능7.1(구글로 로그인)
역할:구글계정을 이용해서 로그인한다
입력:구글로 로그인 버튼 클릭
출력:구글 계정 입력창 출력
기능7.2(이메일로 로그인)
역할:이메일로 가입한 계정으로 로그인
입력:이메일로 로그인 버튼 클릭
출력:이메일 계정 입력창 출력
기능7.3(로그인 버튼)
역할:비밀번호가 맞다면 로그인
입력:계정과 비밀번호
출력:성공 - 어플 메인 화면으로 이동
실패 - 뭐가 틀렸는지 메세지 출력
